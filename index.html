Core question this iteration answers
How does one recipe serve many batch sizes & pack sizes?  →  A recipe has versions; each version has multiple batch configurations; each batch has multiple packaging configurations; each packaging links to a product/variant. Publishing maps version + batch + packaging to each variant. Walk it on the Batch & Packaging screen.
